Tema 18: Polimorfizmi dhe Trashegimia

Polimorfizmi eshte nje teknike ne programimin e orjentuar drejt objekteve dhe ofron perparesi ndaj programit funksional. Polimorfizmi mundeson perdorimin e nje metode tek objekte te klasave te ndryshme.

Shembull:

class Qeni:
    def ze(self):
        print("Qeni leh")

class Macja:
    def ze(self):
        print("Macja mjaullin")

kafsha1 = Qeni()
kafsha2 = Macja()

kafsha1.ze()
kafsha2.ze()

Trashegimia eshte nje koncept i OOP qe na lejon te krijojme klasa te reja bazuar ne klasa ekzistuese, duke trasheguar atributet dhe metodat e tyre. Trashegimia eshte vetia e nje klase per te trasheguar vetite dhe metodat e nje klase tjeter. Perfaqeson mire mardheniet ne boten reale. Siguron riperdorimin e nje kodi. Ka natyre kalimtare.

Sintaksa:

class klasa_Prind:
    #blloku i kodit

class klasa_Femije(klasa_Prind):
    #blloku i kodit

Shembull:

class TeDhenat:
    def __init__(self, emri, mosha):
        self.emri = emri
        self.mosha = mosha

    def te_dhenat(self):
        print(self.emri, self.mosha)

class Personi(TeDhenat):
    pass

p1 = Personi("Gezim", 15)
p1.te_dhenat()
