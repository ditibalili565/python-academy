Tema 10: Vargjet e karaktereve

String eshte tipi i te dhenes qe perdoret per te ruajtur text. Nje string eshte nje sekuence karakteresh qe vendoset brenda thonjezave teke, dyshe ose te trefishta.

Shembull:

print('Shkolla "Industriale" ')

Karakteristikat:

  • Cdo karakter ka nje index
  • Indeksimi fillon nga 0
  • Ka dhe indeksim negativ

Per te gjetur gjatesine e nje stringu perdorim funksionin len().
Per te kthyer nje string ne shkronja te medha perdorim funksionin upper().
Per te kthyer nje string ne shkronja te vogla perdorim funksionin lower().
